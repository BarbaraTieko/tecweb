# Questão 1

Implemente uma página nova para o seu **Projeto 1A**. Essa página deverá estar disponível ao acessar a rota "tecweb".

Ou seja, essa nova página estará disponível no endereço: http://0.0.0.0:8080/tecweb

Essa página deve obrigatoriamente conter uma imagem. Você pode utilizar o HTML e a imagem disponíveis aqui. Mas se preferir tem liberdade para colocar outra imagem e modificar o HTML.

A seguinte rubrica será utilizada para corrigir esta questão:

- I: Não entregou;
- D: Entregou, mas o código não pode ser executado (ocorre algum erro que não permite que seja realizado nenhum teste);
- C: É possível acessar a página na rota definida, mas a imagem não carrega ou não é apresentada corretamente;
- B: É possível acessar a página na rota definida e a imagem é apresentada corretamente;
- A: É possível acessar a página na rota definida, a imagem é apresentada corretamente e o texto e a imagem estão centralizados horizontalmente na tela.
- A+: Existe algum botão/link na página principal do projeto 1A que leva a nova página.



# Questão 2

Baseado no que você aprendeu com o Projeto 1B, crie um novo projeto em Django chamado insperpessoa. O insperpessoa é um sistema simples de cadastro de pessoas, neste cadastro o sistema armazena o nome e o sobrenome da pessoa.

Além disso, ele mostra a lista de pessoas e a quantidade total de pessoas cadastradas no sistema.

Dica: https://docs.djangoproject.com/en/3.1/ref/models/querysets/#django.db.models.query.QuerySet.count

Você pode se basear no código do Projeto 1B se preferir, mas pode ser mais fácil criar um projeto do zero.

A seguinte rubrica será utilizada para corrigir esta questão:

- I: Não entregou;
- D: Entregou, mas o código não pode ser executado (ocorre algum erro que não permite que seja realizado nenhum teste);
- C: É possível acessar uma página que mostra um formulário, mas a implementação de criação **ou** apresentação da quantidade de pessoas cadastradas não funciona;
- B: É possível acessar uma página que mostra um formulário **e** a quantidade de pessoas cadastradas. Além disso, é possível criar novas anotações;
- A: Atingiu o conceito B e lista as pessoas cadastradas.
- A+: Implementou o estilo da página
