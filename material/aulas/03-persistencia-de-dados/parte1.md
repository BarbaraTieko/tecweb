# Parte 1: Instalando o SQLite

Tecnicamente não é necessário instalar o SQLite, pois o Python já vem com uma versão do SQLite instalada. Para testar execute o comando

```shell
$ python -c 'import sqlite3'
```

no seu terminal. Se não der nenhum erro está tudo certo.

Nós vamos instalar também o DB Browser for SQLite, um programa que utilizaremos para visualizar o nosso banco de dados. Você não precisa dele para que o SQLite funcione. É apenas um programa que vai auxiliar a visualização do que está acontecendo. Faça o download e instale o programa [clicando neste link e seguindo as instruções](https://sqlitebrowser.org/dl/).

Depois de instalar o programa, siga para a [parte 2](parte2.md).
