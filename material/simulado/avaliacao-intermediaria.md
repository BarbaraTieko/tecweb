# Questão 1

Implemente uma página de erro de página não encontrada para o seu **Projeto 1A**. Essa página deve ser mostrada para **qualquer** endereço acessado que seja diferente das rotas válidas (páginas ou recursos como arquivos CSS, JS e imagens).

Essa página de erro deve obrigatoriamente conter uma imagem. Você pode utilizar o HTML e a imagem disponíveis aqui.

[clique aqui para baixar o HTML](../arquivos/404.html)

[clique aqui para baixar a imagem](../arquivos/meme.gif).

Mas se preferir tem liberdade para colocar outra imagem e modificar o HTML.

# Questão 2

Baseado no que você aprendeu com o Projeto 1B, crie um novo projeto em Django chamado Memo. O Memo é um sistema simples de anotações que sempre mostra **apenas** a última anotação criada. Ou seja, quando o usuário cria uma nova anotação ele **não terá mais acesso às anotações antigas**.

Dica: https://docs.djangoproject.com/en/4.1/ref/models/querysets/#last

Você pode se basear no código do Projeto 1B se preferir, mas pode ser mais fácil criar um projeto do zero.
